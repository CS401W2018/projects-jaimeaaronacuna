What is the difference between an ID and a Class?

With an ID each tag can only have one and no two or more IDs can share the same name, While a Class can have more than one per tag and multiple tags can share the same class name.

What is the difference between a SPAN tag and a DIV tag?

A SPAN tag is more specific and cannot have other tags inside it and it also does not cause line breaks, while a DIV tag is for a wider scope which accepts other tags inside it and it also causes line breaks.
