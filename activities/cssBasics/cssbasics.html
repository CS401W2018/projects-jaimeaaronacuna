CSS Basics

Explain the difference between inline, embedded, and external CSS.

Inline CSS is put in the specific tags that is being designed.
Embedded CSS is coded in a desegnated style tag in the head tag.
External CSS is coded on a different file and is linked from the html file.

Explain why you should only use external CSS.

External CSS saves a lot of time in working with big websites and it also allows designers to have a desegnated file to work on without the need to be proficient in html.

What is a selector used for?

The selector is used to specify which tag is going to be affected by the css style block.
