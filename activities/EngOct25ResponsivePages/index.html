Why do we need a viewport meta tag?

To provide a nice experience for people accessing the webpage using a smaller screen device.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The one that is initialized the latest will be the one that will be taken priority.

How do you change the CSS for a web page displayed on a phone?

you add an @media only screen and (max-width: 800px) tag and manipulate the css changes if the screen goes below the max width specified.
