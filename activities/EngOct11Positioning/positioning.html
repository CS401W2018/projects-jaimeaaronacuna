What is the difference between a block and an inline tag? Give examples of each one.

A block takes up the whole width available by default while inline tag only takes up what is required. An example of a block is a dive tag and an example of an inline tag is a span tag.

What is normal flow in HTML?

The default way of how elements of a wep page is arranged.

What does float do? Why do you need to clear the float?

Float positions an element to the right or to the left and wraps other elemetns around it. It is important to clear a float as it will keep on wrapping other elements around it if it is not cleared.

What do absolute, fixed, and relative do?

Absolute removes the elemement from the normal flow and makes other elements act like it does not exists. Fixed does the same thing but positions the element to be fixed to the viewport. And relative moves the element a number of units away relative to its original position and does not remove the element from the normal flow.

How can you make nice navigation buttons for mobile devices?
