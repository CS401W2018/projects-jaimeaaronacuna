Internet

A global system of computers interconnected with eachother.

Web

The Web or World Wide Web (WWW) is a system of information sharing through the internet that is meant to be user-friendly.

How are the Internet and Web different?

The Web is the system of sharing information while the Internet is the connectivity that is used as the medium for the Web.

ASCII and Unicode

ASCII stand for American Standard Code for Information Interchange. It is a digital coding system that represents american used letters, numbers, and symbols for standard use. And Unicode is the Universal version which supports symbols used by different countries.

What is UTF-8

Stands for Unicode Transform Format - 8-bit. It is a version of Unicode that is encoded in four to eight byts for backwards compatibility with ASCII.

URL

Stands for Uniform Resource Locator. It is an address used to access specific files or resources on the internet.

Domain Name

It is a user-friendly name assigned to a specific IP for a better user experience. Examples of these are google.com which represents the IP 142.250.69.238 and facebook.com which represnets the IP 31.13.70.36.

HTTP

Stands for Hypertext Transfer Protocol. It is a protocol used by clients and servers for communication through the internet.

HTTPS

Stands for Hypertext Transfer Protocol Secure. An encrypted version of HTTP for a more secure connection.

IP Address

Stands for Internet Protocol Address. It is special numerical address assigned to a computer that uses the internet for accurate point of end to end communication. There are two versions of IP Address: IPV4 and IPV6.

DNS

Stands for Domain Name System. It is a system that translate domain names into IP addresses to be able to be read by a computer.

W3C Recommendations

Stands for World Wide Web Consortium Recommendations. It is the official standard and guidlines for web development. It is developed by the W3C, an international organization that aims to standartize web development.

XML, XHTML, HTML, HTML5

These are different programming languages with different use cases used for web development.

XML (Extensible Markup Language) is used to define and structure data.
XHTML (Extensible Hypertext Markup Language) is a version of HTML that follows the rules of XML.
HTML (Hypertext Markup Language) is used for displaying contents to the user.
HTML5 is the latest significant version of HTML.
